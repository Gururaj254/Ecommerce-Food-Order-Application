Project Title: 
# Ecom Food Delivery Microservices

## Services
- Eureka Service Registry
- API Gateway
- Restaurant Service
- Menu Service
- Order Service

## Tech Stack
- Java
- Spring Boot
- Spring Cloud (Eureka, Gateway)
- feign client
- Resillence4J Circuit Breaker
- spring actuator 
- REST APIs
- MySQL

## Execution Order

Eureka Service Registry
API Gateway
Restaurant Service
Menu Service
Order Service

Project Description:
Microservices-based food ordering system with Restaurant, Menu, and Order services. Full service discovery, gateway routing, and end-to-end order management.

Features:

Add/View Restaurants

Add/View Menu Items per Restaurant

Place/View Orders (with automated price & item name fetching)

Service discovery with Eureka

API routing via Spring Cloud Gateway

Inter-service communication with Feign clients
